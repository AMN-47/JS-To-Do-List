# JS-To-Do-List
## Description
This is a todo list application created using JavaScript. 

This follows the Javascript coursework provided by the OdinProject.

This project was aimed to allow me to practice object oriented principles further through applying them in this project.

## Skills Learnt
- Use of DOM Manipulation to create HTML Elements
- Styling Java Script Created Elements within the JavaScript file
- Object oriented programming through calling of functions from one file to another as well as one function to another.
- Styling using Claude AI and CSS
